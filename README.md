# React Redux Dev

## Start  
* `npm install`
* `npm run dev`
* `npm run build-dev`
* `npm run build`


## Tech stack  
* React
* Redux
* ImmutableJS
* Express
* Webpack2


## Features  
* HMR
* ES6
* Eslint
* Commons Chunk
* Extract CSS
* Sass
* CSS Autoprefixer
* HTML Loader
* HTML Template
* Babili
* Redbox
